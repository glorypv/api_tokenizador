## Start project
```bash
$ run.sh
```

## Start Apps
```bash
docker-compose up -d
```

# Instala las dependencias
```bash
npm install
```

# Ejecuta las migraciones de Prisma
```bash
npx prisma migrate dev --name "init"
```

# Inicia el proyecto
```bash
npm run start
```

## Test
# Ejecutar test
```bash
npm run test
```
